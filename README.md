# ❌⭕ Tic-Tac-Toe Game

An interactive, responsive, and modern web-based Tic-Tac-Toe game built to practice advanced JavaScript game logic, state tracking, and dynamic UI updates.

---

## 🎯 Project Objective
The goal of this project was to implement a complete game loop in JavaScript. It demonstrates my understanding of event listeners, array-based win-condition tracking, and handling user turns (Player X vs. Player O) dynamically in the DOM.

---

## ✨ Key Features
* 🎮 **Smooth Gameplay** – Seamless turn-based switching between Player X and Player O.
* 🏆 **Win & Draw Detection** – High-efficiency logic that instantly checks for winning combinations or tie games.
* 🔄 **Instant Restart** – A "Reset Game" button to clear the board and start a new match instantly without refreshing the page.
* 🎨 **Interactive UI** – Hover effects on grid cells and clear visual indicators showing whose turn it is.
* 📱 **Mobile Friendly** – Fully responsive grid layout that auto-scales perfectly on mobile screens.

---

## 🛠️ Built With
* **HTML5** – For the game grid structure and control elements.
* **CSS3** – Using Grid/Flexbox for the game board layout and modern animations.
* **JavaScript (Vanilla)** – For implementing the core game logic, win-combination arrays, and status updates.
